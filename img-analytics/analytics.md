# requirement

上传一个图片文件，分析内容，最后输出数据表格。

## 选择日期

选择上传图片的日期，默认是昨天的日期。

## 上传文件

提供一个输入，可以上传一个图片文件。

文件的内容是Pacer里面的群组的跑步排名。有名次，姓名，步数 3列。

示例图片：
![示例图片](./result.jpg "result screenshot")

## OCR

上传文件有，前端进行OCR，然后提取出文本内容，最后丢给copilot ai进行分析补全数据。最后copilot ai给出补全后的数据表格。

每一行是一个队员和步数。

## 组员

有输入框，可以输入所有组员，比如
```
name2
name3
name1
```

## 输出

输出表格。表格大概如下

2026年3月20日

|member name|steps|
|-----------|-----|
|name1|12322|
|name2|8889|
|name3|8769|

点击表头`steps`可以复制整列的数据，用于后面粘贴到Excel。

输出的表格数据顺序，需要按照上面输入组员名字的顺序排列。

用户名一定要对应上，copilot ai在分析时，需要一一对应组员的名字，可以智能的匹配上。

匹配不上的用户，需要把文字设置为黄色warnning状态，提醒用户自己去匹配。

输出的行数与OCR结果一致。不要删除数据。

## 过程日志

每一步操作的过程都要在页面上显示日志，好让操作者知道目前在做什么。

## 后台服务
调用服务在server.mjs中。调用服务的过程， 可能需要使用代理服务器来访问网络。

代理的地址，由server.mjs从系统环境参数读取，需要代理地址，用户名和密码。

如果系统环境参数读取不到，则表示不需要使用代理，直接访问网络。

前端页面请给出系统代理设置example。
